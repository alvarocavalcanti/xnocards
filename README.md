# X, N & O Cards

An anonymous TTRPG safety tool extension for [Owlbear Rodeo](https://www.owlbear.rodeo/).

X, N, and O Cards are check-in tools for tabletop RPG sessions:

- **X Card** — Stop or change the current content immediately.
- **N Card** — Slow down or fade to black; content is heading somewhere uncomfortable.
- **O Card** — Group check-in: is everyone comfortable to continue?

Any participant can play a card anonymously. The card appears as a full-screen overlay for all connected players.

## Install

Add the extension via the Owlbear Rodeo extension store or manually using the manifest URL:

```
https://xnocards.vercel.app/manifest.json
```

## Development

```bash
npm install
npm run dev
```
